# Home-Lab-series
Self-study Homelab Series

This repository contains all documentation, slides, configs, and other materials for 10-week Homelab Series for the student aspired to build their  personal lab.

The goal of this series is to help students build practical, hands-on skill in homelabbing using tools that can be replicated on personal hardware or infrastructure.

Series Overview

Weeks and Topic*

    Homelab Inspiration and VMware Basics
    Building Your First Debian Server
    Hosting Your Own Minecraft Server
    Internal Website Hosting with Apache2
    Access your homelab from anywhere (Tailscale Setup)
    DNS Sinkhole (Pi-Hole Setup)
    Set Up Your Own Cloud (Samba and NAS)
    SIEMS and Logging
    Monitoring and Dashboards
    Homelab Capstone Showcase

Each week includes:

    A lab guide ('docs/weekXX/guide.md')
    Optional instructor notes for the presenter
    Slides ('slides/weekXX/')
    Config files or scripts when applicable

Requirements

To follow along with most labs, you will need one of:

    A computer capable of running VMware Workstation or
    Access to DePaul's SecLabs remote VMs (instructions provided separately)

Some labs may also use:

    Debian, PI OS, Ubuntu Server IOSs
    TrueNAS ISO
    Basic familiarity with the command line
